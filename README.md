Puls4 XBMC Addon
=======
Puls4 is an addon that gives you access to the Puls4 Video Platform.


Requirements
------------
Tested on XBMC Eden/Frodo


Supported platforms
-------------------
Windows, Linux , Android and OSX


Current Features
----------------
* All Shows
* Search
* H264 Stream
* Top Shows and Most Popular


Missing Features
----------------
* Date Search Function
* Pager


Known Issues
------------
* None so far
